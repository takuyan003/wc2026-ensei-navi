# W杯遠征ナビ 2026

**北中米W杯（2026年6月11日〜7月19日）に現地参戦する日本人サポーターのための観戦ガイドアプリ**

非公式の個人制作ファンガイド。FIFAおよび各団体とは関係ありません。

---

## プロジェクト概要

| 項目 | 内容 |
|---|---|
| ターゲット | 北中米W杯に現地参戦する日本人（一般公開想定） |
| 形態 | 静的Webアプリ（PWA）。フレームワーク不使用・ビルド不要 |
| 対応 | スマホ最適化（モバイルファースト）、オフライン動作対応 |
| 状態 | **プロトタイプ完成・ファクトチェック済み・デプロイ待ち** |
| 制作開始 | 2026年6月12日（大会開幕翌日） |

### なぜこのアプリか

- 開催が米国11・メキシコ3・カナダ2の計16都市に分散し、移動・時差・通貨・チップ文化が国ごとに違う
- 日本はグループFで **ダラス → モンテレイ → ダラス** と国境をまたぐ日程
- 現地は通信が不安定な場面が多く、オフラインで動く実用ツールに需要がある

---

## 機能（v1.0 — 5タブ＋5つの行動モード）

1. **今日** — 次の日本戦カウントダウン、会場現地時間と日本時間の時計、今日やること（試合日に応じて変化）、チェックリスト進捗、クイックアクション（チケット/Maps/各モード起動）
2. **日本代表** — グループF全3戦を現地時間・日本時間の並記で表示。放送局、ノックアウト日程、**勝ち点シミュレーター**（勝/分/負を選んで突破可能性を判定）
3. **スタジアム** — 16会場のガイド（アクセス🚇・暑さ🌡️・夜の治安🌙・持ち込み🎒・帰り方🔙）。各会場の現在時刻、国別フィルタ、日本戦バッジ、Google Maps/公式情報ボタン
4. **チップ計算** — 米・加・墨×シーン別の相場で計算、円換算付き。為替はオンライン時に自動取得、オフライン時は保存済みレート
5. **遠征メモ** — チェックリスト（保存式）、マイ情報登録、緊急通報911、ESTA/eTA/FMM、暑さ対策、持ち込み規定、フレーズ集、公式リンク集

### 行動モード（現地で焦った時に押すボタン）

- 🎒 **試合前モード** — 出発前の手順6ステップ
- 🎫 **入場前モード** — ゲート前の手順5ステップ
- 🏨 **試合後帰宅モード** — 「すぐUberを呼ばない」等の帰宅手順＋ホテル住所表示＋見せる英語/スペイン語
- 🤝 **はぐれた時モード** — 集合場所・同行者メモ表示＋見せる英語/スペイン語
- 🚨 **緊急時モード** — 911発信ボタン＋「日本人です・通訳が必要です」の見せる文＋保険会社メモ

すべて番号付き手順・大きなボタン・大きな文字で、焦っていても読めるデザイン。

### 保存機能（localStorage・端末内のみ）

チェックリスト完了状態 / ホテル名・住所 / 集合場所 / 同行者メモ / 緊急連絡先 / 為替レート / シミュレーターの選択

## ファイル構成

```
wc2026-ensei-navi/
├── index.html      # エントリーポイント（タブUI骨格）
├── styles.css      # スタイル（サムライブルー基調・モバイルファースト）
├── app.js          # 描画ロジック・チップ計算・為替取得・SW登録
├── data.js         # 試合日程・16会場データ・チップ相場（★更新はここ）
├── sw.js           # Service Worker（オフライン対応・現在 v3）
├── manifest.json   # PWAマニフェスト
├── icon.svg        # アプリアイコン
├── FACTCHECK.md    # ファクトチェック記録（45項目・検証済み）
└── README.md       # このファイル
```

## ローカルでの動かし方

```bash
cd wc2026-ensei-navi
python3 -m http.server 8765
# → http://localhost:8765 を開く
```

※ Service Workerがキャッシュするため、ファイル更新が反映されないときは
`sw.js` の `CACHE` バージョンを上げる（v3 → v4）。
